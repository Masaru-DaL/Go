## 1. Introduction

#### 1-1. GraphQLの特徴

* RESTよりも効率的で強力かつ柔軟なAPIを提供する**新しいAPI規格**
* **Facebookによって開発、オープンソース化**
* クライアントがAPIから必要とするデータを正確に指定する事ができる
  + 宣言的なデータ取得
* GraphQLサーバは、**単一のエンドポイントのみ**を公開し、クライアントが要求したデータで正確に応答する

#### 1-2. APIのためのクエリ言語

GraphQLはデータベース技術であると混同されるが、誤解である。
**GraphQLはAPIのためのクエリ言語**。
